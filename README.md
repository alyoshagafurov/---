# Сайт публикаций

Сайт с публикациями (аналог 999.md): единая структура публикации (Название, Описание, Фото)
независимо от категории. Регистрация и вход, модерация публикаций через админ-панель,
личные сообщения, избранное, жалобы, настройки профиля. Адаптивная вёрстка
(ПК / планшет / телефон). Интерфейс — на русском языке.

Стек: **Python + Django 5.2 (LTS)**, база данных **SQLite** (легко переключается на
PostgreSQL), шаблоны Django, собственный CSS и небольшой ванильный JS.

---

## Быстрый старт (локально)

```bash
# 1. Окружение и зависимости
python3 -m venv .venv
source .venv/bin/activate          # Windows: .venv\Scripts\activate
pip install -r requirements.txt

# 2. База данных
python manage.py migrate

# 3. Демо-данные (категории, пользователи, публикации) — по желанию
python manage.py seed_demo

# 4. Свой администратор (если не используете демо)
python manage.py createsuperuser

# 5. Запуск
python manage.py runserver
```

Открыть: http://127.0.0.1:8000/

Демо-доступ после `seed_demo`:
- **Администратор:** `admin@example.com` / `admin12345`
- **Пользователи:** `user1@example.com` … `user3@example.com` / `user12345`

Админ-панель сайта (по ТЗ): **/panel/**
Стандартная админка Django (резервная): **/django-admin/**

## Настройка почты

По умолчанию письма выводятся в консоль (режим разработки). Для отправки реальных
писem задайте переменные окружения (SMTP):

```bash
export EMAIL_HOST=smtp.example.com
export EMAIL_PORT=587
export EMAIL_HOST_USER=login
export EMAIL_HOST_PASSWORD=secret
export EMAIL_USE_TLS=true
export DEFAULT_FROM_EMAIL="Публикации <no-reply@example.com>"
```

Письма по ТЗ: регистрация, восстановление пароля (ссылка действует 1 час и
становится недействительной после смены пароля), смена пароля, смена почты.

---

## Полезные переменные окружения (для боевого режима)

| Переменная | Назначение | По умолчанию |
|------------|-----------|--------------|
| `DJANGO_SECRET_KEY` | секретный ключ | небезопасный для разработки |
| `DJANGO_DEBUG` | режим отладки | `True` |
| `DJANGO_ALLOWED_HOSTS` | хосты через запятую | `*` |
| `DJANGO_CSRF_TRUSTED_ORIGINS` | доверенные origin'ы | — |
| `SITE_NAME` | название сайта / логотип | `Публикации` |
| `DJANGO_TIME_ZONE` | часовой пояс | `Europe/Moscow` |

Перед публикацией: `DJANGO_DEBUG=false`, задать `DJANGO_SECRET_KEY`,
`DJANGO_ALLOWED_HOSTS`, выполнить `python manage.py collectstatic`.

## Структура проекта

```
config/      — настройки и корневые URL
accounts/    — пользователи, вход/регистрация, восстановление, профиль, настройки, избранное
listings/    — категории, публикации, фото, избранное, жалобы
messaging/   — диалоги и сообщения
panel/       — админ-панель сайта (модерация, пользователи, жалобы, категории, правила)
core/        — главная, правила, контакты, тексты сайта, отправка писем
templates/   — HTML-шаблоны
static/      — CSS и JS
media/       — загруженные файлы (аватары, фото публикаций)
```
